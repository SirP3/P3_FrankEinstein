# YTM Source Folder Policy

All source files for YTM runs must stay local-only under:

output/youtube_mining/<run_id>/source/

Allowed source examples:
- .md manual source notes
- .txt transcript text
- .vtt transcript files

Forbidden:
- committing raw transcripts
- committing VTT/TXT source files
- committing source-derived radar cards or handoff outputs

Current status: policy only; source import script not implemented yet.
